# Manuscript Source Snapshot

The files in this directory are retained for provenance and manuscript-build reproducibility. They correspond to the pre-submission LaTeX source snapshot distributed with the GitHub `v1.0.0` release and archived on Zenodo at [10.5281/zenodo.21331678](https://doi.org/10.5281/zenodo.21331678).

This directory is not a verbatim copy of the current journal-specific Word manuscript. Journal-specific title wording, author metadata, declarations, narrative editing, and document formatting may differ from this archived source snapshot. Those presentation and metadata differences should not be interpreted as changes to the underlying scientific analysis.

For computational reproducibility, use the versioned analysis scripts, configuration, machine-readable result tables, source-provenance documentation, and validation outputs in the repository. The GitHub release and Zenodo deposit remain frozen historical records; later maintenance on the `main` branch does not rewrite those published artifacts.
